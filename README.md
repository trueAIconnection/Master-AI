# Master AI Knowledge Base

## Purpose
Centralized knowledge repository for AI Connection's SaaS development projects. This repository serves as the single source of truth for:
- Project documentation and context
- Reusable prompt templates
- Development templates and boilerplates
- Learning resources and best practices
- Technical documentation

## Structure

### `/projects/`
Individual SaaS project folders, each containing:
- Project context and requirements
- Custom prompts specific to the project
- Development notes and decisions
- API documentation

### `/master-prompts/`
Reusable, battle-tested prompt templates organized by category:
- Engineering prompts
- Business strategy prompts
- API integration prompts
- Content creation prompts

### `/templates/`
Project templates and boilerplates:
- SaaS project starter templates
- API integration patterns
- Documentation templates
- Workflow templates

### `/documentation/`
Guides, tutorials, and reference materials:
- Setup guides
- Best practices
- Technical documentation
- Integration guides

### `/learning/`
Educational content and resources:
- Claude API learning materials
- Prompt engineering techniques
- SaaS development resources
- No-code/low-code platform guides

## Usage

### For Claude Accounts
Both Claude accounts (trueAIconnection and marketing@epicgrowth.com) have access to this repository through MCP.

**Example commands:**
```
"Read my master engineering prompts"
"Show context for [project-name]"
"Add this prompt to master-prompts"
```

### Multi-Account Sync
This repository automatically syncs across:
- Computer 1 (primary development)
- Computer 2 (secondary/testing)
- Both Claude accounts

## Maintenance
- Update prompts when you discover better versions
- Add new projects as they're created
- Keep documentation current
- Archive completed projects
