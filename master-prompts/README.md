# Master Prompts Directory

## Purpose
Reusable, high-quality prompt templates that have proven effective across multiple projects.

## Categories

### Engineering Prompts
- API integration patterns
- Code review and debugging
- Architecture design
- Testing strategies

### Business Prompts
- Market research
- Competitive analysis
- Business model validation
- Pricing strategy

### API Prompts
- Claude API integration
- Authentication patterns
- Rate limiting strategies
- Error handling

### Content Creation
- Marketing copy
- Documentation writing
- User interface text
- Email templates

## Format
Each prompt template should include:
1. **Purpose** - What the prompt accomplishes
2. **Context Variables** - What information needs to be filled in
3. **Template** - The actual prompt text
4. **Example Usage** - A concrete example
5. **Expected Output** - What kind of response to expect

## Best Practices
- Test prompts thoroughly before adding them here
- Include clear instructions and context
- Document which projects used each prompt successfully
- Update prompts when you discover improvements
- Note any Claude model-specific considerations

## Usage with Claude
```
"Show me my engineering prompts"
"Add this prompt to master-prompts"
"Which prompt should I use for [task]?"
"Update the [category] prompts with this improvement"
```
